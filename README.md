
# Cicatrizes Invisíveis

Cicatrizes Invisíveis é um projeto web desenvolvido com foco em **conscientização sobre o abuso psicológico no ambiente familiar**, utilizando a tecnologia como meio de reflexão e expressão narrativa.

O projeto foi criado como parte de um trabalho acadêmico, unindo **desenvolvimento web** e **produções literárias autorais**.

---

## 🎯 Proposta

A aplicação apresenta conteúdos informativos e obras literárias em formato digital, abordando temas como:
- abuso psicológico
- identidade
- pertencimento
- impactos emocionais no núcleo familiar

Todo o conteúdo é disponibilizado por meio de uma experiência web simples e acessível.

---

## 🛠️ Tecnologias Utilizadas

- HTML  
- CSS  
- TypeScript
